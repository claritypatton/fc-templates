# Template preview images

These PNGs are what AMs see in the Web Studio template-picker modal. They're served via the GitHub raw URL referenced in `templates.json`.

## Current state: placeholders

The PNGs in this directory right now are **placeholders** generated by `_generate-placeholders.py`. They use each template's actual color palette so the picker shows brand-correct identity, but they are not real screenshots of the templates.

The watermark band at the bottom of each placeholder reads "PLACEHOLDER PREVIEW · REPLACE BEFORE LAUNCH" — that's intentional. They should be replaced with real screenshots before AMs use the picker on real engagements.

## Replacing with real screenshots

For each template:

1. `cd templates/{template-id}`
2. `npm install && npm run dev`
3. Open the local site in a browser at the default state (the design-time placeholder content in `lib/site-config.ts`)
4. Take a screenshot of the home page at desktop size — 1200×800 PNG is the target dimensions
5. Save as `previews/{template-id}.png` (overwriting the existing placeholder)
6. Commit. The picker shows the new image within ~5 minutes (Andar's `/api/templates` cache TTL).

## File requirements

- Filename matches the template's `id` field exactly (lowercase, no spaces)
- PNG format (`.png`)
- Roughly 1200×800 pixels (3:2 aspect) — picker UI scales down, but that's the source resolution
- Reasonable file size (under 500 KB each, ideally under 100 KB)

## Adding a fourth template's preview

1. Drop `previews/{new-id}.png` into this directory
2. Add the entry to `templates.json` with `previewImage` pointing at the GitHub raw URL for the new file
3. CI's "Every active template has a preview image" check enforces that this file must exist for any active template

## The placeholder generator script

`_generate-placeholders.py` is committed for reproducibility — if a new template is added and a real screenshot isn't ready yet, run the script after adding the new template's palette to its `TEMPLATES` dict to generate a brand-matching placeholder. Replace later.
